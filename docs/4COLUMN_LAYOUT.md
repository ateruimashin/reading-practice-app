# Reading Practice App - 4カラムレイアウト実装完了

## 変更内容サマリー

3カラムレイアウトから4カラムレイアウト（VSCode/IntelliJスタイル）に変更しました。

## 新しいレイアウト構成

```
┌────┬──────────┬──────────┬──────────┬──────────┐
│Icon│ Sidebar  │ Source   │ Resize   │ Trans.   │
│Bar │(可変幅)   │ Editor   │ Handle   │ Editor   │
│    │          │(可変幅)   │          │(可変幅)   │
│48px│ 280px    │ 500px    │  4px     │ 残り全部  │
└────┴──────────┴──────────┴──────────┴──────────┘
```

### カラムの役割

1. **アイコンバー (48px固定)**
   - 📄 Documents アイコン
   - 📚 Vocabulary アイコン
   - リサイズ不可

2. **サイドバー (可変幅、表示/非表示)**
   - アイコンバーで選択されたビューを表示
   - Documents: 文書一覧
   - Vocabulary: 単語帳一覧

3. **元テキストエディタ (可変幅)**
   - ソーステキストの編集

4. **翻訳エディタ (残り全幅)**
   - 翻訳テキストの編集

## 実装された機能

### ✅ アイコンバーの動作

1. **アイコンをクリックした時の挙動**
   - サイドバーが閉じている → サイドバーが開き、対応するビューを表示
   - 同じアイコンを再クリック → サイドバーが閉じる
   - 別のアイコンをクリック → サイドバーは開いたまま、ビューが切り替わる

2. **ビジュアルフィードバック**
   - アクティブなアイコンは左側に青いボーダー表示
   - ホバー時は背景色が変化

### ✅ サイドバーの動作

1. **表示制御**
   - アイコンバーの状態に連動
   - 閉じる/開くボタンは廃止
   - アニメーションなしで即座に切り替わる

2. **リサイズ**
   - サイドバーが開いている時のみリサイズ可能
   - 最小幅: 250px
   - デフォルト幅: 280px

### ✅ 単語帳の統合

- メニューバーのVocabularyボタンを削除
- サイドバー内で単語帳を表示
- モーダルではなく、インラインでの表示に変更
- レイアウトを縦方向に最適化（カードを縦に並べる）

## 主な技術的変更点

### 1. アイコンバーコンポーネント

```typescript
export type SidebarView = 'documents' | 'vocabulary' | null;

interface IconBarProps {
  activeView: SidebarView;
  onViewChange: (view: SidebarView) => void;
}
```

- アクティブなビューを管理
- クリック時のトグル動作を実装

### 2. MainLayoutの状態管理

```typescript
const [activeSidebarView, setActiveSidebarView] = useState<SidebarView>('documents');
```

- サイドバーの表示/非表示とビューを1つの状態で管理
- `null` = サイドバー非表示
- `'documents'` または `'vocabulary'` = 対応するビューを表示

### 3. 条件付きレンダリング

```typescript
{activeSidebarView && (
  <>
    <div className="sidebar-column">...</div>
    <div className="resize-handle">...</div>
  </>
)}
```

- サイドバーとリサイズハンドルを条件付きでレンダリング
- パフォーマンスとレイアウトの安定性を向上

## CSS変数の活用

すべてのコンポーネントでCSS変数を使用し、ダークモードに完全対応：

- `--main-background`: メイン背景色
- `--sub-background`: サブ背景色
- `--border-color`: ボーダー色
- `--text-primary`: メインテキスト色
- `--text-secondary`: セカンダリテキスト色
- `--primary-color`: プライマリカラー
- `--hover-background`: ホバー時の背景色

## テスト方法

1. **アイコンバーのテスト**
   ```
   - 📄アイコンをクリック → 文書一覧表示
   - 📚アイコンをクリック → 単語帳表示に切り替わる
   - 📚アイコンを再クリック → サイドバーが閉じる
   ```

2. **リサイズのテスト**
   ```
   - サイドバーとエディタの境界をドラッグ → 幅が変わる
   - サイドバーを閉じる → リサイズハンドルが消える
   ```

3. **ダークモードのテスト**
   ```
   - Theme: Light → すべて明るい色
   - Theme: Dark → すべて暗い色
   - アイコンバーも自動的に色が変わる
   ```
