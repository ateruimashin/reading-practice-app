# Reading Practice App - 単語帳機能

## 実装済み機能

### 1. 単語帳追加機能
- エディタ内でテキストを選択すると、コンテキストメニューが表示されます
- 「📚 Add to Vocabulary」をクリックすると、単語帳追加モーダルが開きます
- 以下の情報を入力できます：
  - **Term**: 単語またはフレーズ（必須）
  - **Meaning**: 意味や翻訳（必須）
  - **Context**: 追加のコンテキストやメモ（任意）
  - **Add as example**: 選択したテキストを例文として追加（任意）

### 2. 単語帳一覧表示
- メニューバーの「📚 Vocabulary」ボタンをクリックすると単語帳一覧が表示されます
- 以下の機能があります：
  - **言語フィルター**: 特定の言語の単語のみを表示
  - **検索**: 単語を検索
  - **カード表示**: 単語がカード形式で表示されます

### 3. 単語詳細表示・編集
- 単語カードをクリックすると詳細モーダルが開きます
- 以下の機能があります：
  - **単語の編集**: 単語自体を編集可能
  - **意味の表示・編集・削除**: 複数の意味を管理
  - **例文の表示・削除**: 意味ごとに例文を管理
  - **新しい意味の追加**: 単語に新しい意味を追加

## 使い方

### 単語を追加する方法
1. 文書を開き、エディタ内でテキストを選択します
2. 選択したテキストの近くにコンテキストメニューが表示されます
3. 「📚 Add to Vocabulary」をクリックします
4. モーダルで必要な情報を入力します
5. 「Add to Vocabulary」ボタンをクリックして保存します

### 単語帳を確認する方法
1. メニューバーの「📚 Vocabulary」ボタンをクリックします
2. 単語帳一覧が表示されます
3. 言語フィルターや検索機能を使って目的の単語を探します
4. 単語カードをクリックすると詳細が表示されます

### 単語を編集する方法
1. 単語帳一覧から編集したい単語をクリックします
2. 詳細モーダルで✏️ボタンをクリックして編集モードに入ります
3. 編集後、「Save」ボタンをクリックして保存します

### 単語を削除する方法
- **単語帳一覧から**: 単語カードの🗑️ボタンをクリック
- **詳細モーダルから**: 意味や例文の🗑️ボタンをクリック

## データベース構造

単語帳機能は以下のテーブルで構成されています：

### vocabulary
- 単語の基本情報を保存
- フィールド: id, term, language, created_at, updated_at

### vocabulary_meanings
- 単語の意味を保存（1つの単語に複数の意味を持てる）
- フィールド: id, vocabulary_id, meaning, context, created_at

### vocabulary_examples
- 意味に紐付けられた例文を保存
- フィールド: id, meaning_id, example_sentence, document_id, created_at

## 今後の拡張可能性

以下の機能を追加できます：
- 単語の復習機能（フラッシュカード）
- 習熟度の記録
- エクスポート/インポート機能
- 音声発音の追加
- 画像の追加
- タグ機能

## 開発・実行方法

```bash
# 依存関係のインストール
npm install

# 開発モードで起動
npm run tauri dev

# ビルド
npm run tauri build
```
